# IMPORT-EXPORT-DASHBOARD (POWER-BI)
Developed an interactive Power BI dashboard to analyze a comprehensive dataset of 15,000 global import and export transactions. Utilized data visualization techniques to uncover trade patterns, including country-wise trade volumes, product category trends, shipping method preferences, and port usage.
